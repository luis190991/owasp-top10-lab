# A01 – Control de Acceso Roto: Soluciones

## Reto 1 – IDOR en el Perfil
Inicia sesión como `alice` (sin contraseña) y navega manualmente a:
- `/a01/profile/2` → datos de Bob incluyendo tarjeta de crédito
- `/a01/profile/3` → tarjeta de crédito del Admin: `4333-3333-3333-3333`

**Por qué funciona:** La ruta `profile/<user_id>` obtiene cualquier ID sin verificar
que `session['a01_user']['id'] == user_id`.

**Corrección:**
```python
@a01.route('/profile/<int:user_id>')
def profile(user_id):
    current = session.get('a01_user')
    if not current or (current['id'] != user_id and current['role'] != 'admin'):
        abort(403)
    ...
```

## Reto 2 – Panel de Administración Sin Verificación de Rol
Estando como `alice` (rol=user), navega a `/a01/admin`.
El panel muestra todos los usuarios porque no existe una verificación de `role == 'admin'`.

**Corrección:**
```python
@a01.route('/admin')
def admin():
    user = session.get('a01_user')
    if not user or user['role'] != 'admin':
        abort(403)
    ...
```
