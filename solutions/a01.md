# A01 – Broken Access Control: Solutions

## Challenge 1 – IDOR on Profile
Login as `alice` (no password needed), then manually navigate to:
- `/a01/profile/2` → Bob's data including credit card
- `/a01/profile/3` → Admin's credit card: `4333-3333-3333-3333`

**Why it works:** The route `profile/<user_id>` fetches any ID without checking
`session['a01_user']['id'] == user_id`.

**Fix:**
```python
@a01.route('/profile/<int:user_id>')
def profile(user_id):
    current = session.get('a01_user')
    if not current or (current['id'] != user_id and current['role'] != 'admin'):
        abort(403)
    ...
```

## Challenge 2 – Admin Panel Without Role Check
While logged in as `alice` (role=user), navigate to `/a01/admin`.
The panel renders with all users because there is no `role == 'admin'` guard.

**Fix:**
```python
@a01.route('/admin')
def admin():
    user = session.get('a01_user')
    if not user or user['role'] != 'admin':
        abort(403)
    ...
```

## Challenge 3 – IDOR on Orders
Alice's order is #1. Navigate to `/a01/orders/2` or `/a01/orders/3`.

**Fix:** Check `order['user_id'] == session['a01_user']['id']` before returning.
