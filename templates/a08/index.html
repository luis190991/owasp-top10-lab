Código y datos sin verificación de integridad — deserialización insegura, actualizaciones sin firma.

Retos

¿Cómo funciona el RCE con pickle? El módulo pickle de Python puede serializar objetos arbitrarios, incluyendo aquellos cuyo método __reduce__ ejecuta comandos del sistema al deserializar. Nunca deserialices datos pickle de fuentes no confiables.
