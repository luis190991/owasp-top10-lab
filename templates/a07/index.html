A07:2025 Authentication Failures

Broken implementation of authentication allowing attackers to assume other users' identities.

Challenges
  1. Brute Force: No lockout exists. Script a wordlist attack against the login endpoint.
    Try passwords: password, 123456, sunshine, admin123
  2. Session Prediction: Login once, note your session_id cookie. Session IDs are sequential integers. Try forging a different ID to hijack another session.
  3. Cookie Theft: Login with "Remember Me". Inspect the remember_me cookie — what does it contain?
  4. No Logout Invalidation: Login, copy your session cookie, logout, then manually re-send the old cookie.
