A02:2025 Mala Configuración de Seguridad

Endurecimiento faltante, funciones innecesarias habilitadas, credenciales por defecto, errores verbosos.

Retos
  1. Config expuesta: Accede a /a02/config — encuentra el SECRET_KEY.
  2. Archivo de respaldo: Accede a /a02/backup.cfg — extrae credenciales de base de datos y llaves de API.
  3. Credenciales por defecto: Inicia sesión en /a02/login usando las creds por defecto (admin:admin).
  4. Cabeceras verbosas: Inspecciona los headers de respuesta (DevTools del navegador o curl -I). ¿Qué tecnologías se revelan?
  5. API interna: Llama a /a02/api/data — ¿qué información interna queda expuesta?
