Why This Is Dangerous

No hash or signature is verified before executing the downloaded package. An attacker who compromises plugins.internal (or poisons the DNS) can deliver malicious code that runs with full application privileges.

Fix: always verify sha256 of downloaded artifacts and check a cryptographic signature before executing.
